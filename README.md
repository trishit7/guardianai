# guardianai by Team NEXUS 
**AI-Powered Forensic Asset Protection for Digital Creators**

## 🚀 The Problem
Digital piracy has evolved. Standard copyright bots fail when images are cropped, filtered, or mirrored. Creators lose revenue because traditional "hashing" cannot detect semantic matches.

##  The Solution
GuardianAI (built by **Team NEXUS**) uses **Google Gemini 1.5 Flash** to perform multimodal forensic scans. 
- **Reactive Protection:** Scans two images to detect unauthorized modifications.
- **Proactive Protection:** Embeds steganographic watermarks into pixels.

##  Tech Stack
- **AI Model:** Gemini 1.5 Flash (Multimodal Vision)
- **Frontend:** HTML5 / CSS3 (Cabinet Grotesk Typography)
- **Engine:** JavaScript (Base64 Image Encoding & LSB Steganography)
- **API:** Google Generative AI SDK

##  Installation & Use
1. Clone the repo.
2. Open `index.html` in any modern browser.
3. Obtain a Gemini API Key from [Google AI Studio](https://aistudio.google.com/).
4. Upload an original and a suspected leak to see the **Forensic Report**.
